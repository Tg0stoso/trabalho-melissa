Funcionais

  • O sistema deve registrar quais livros os alunos estão lendo e por quanto tempo
  • O sistema deve conter um ambiente interativo
  • O sistema deve ser online
  • O sistema deve verificar quais livros estão disponíveis para empréstimo
  • O sistema deve permitir a reserva de um exemplar antecipadamente
  • O sistema deve enviar um aviso ao usuário quando a data de devolução estiver próxima
  • O sistema deve disponibilizar livros digitais
  • O sistema deve mostrar recomendações de livros baseados no que o usuário já leu
  • O sistema deve conter um espaço para os usuários discutirem sobre os livros
  • O sistema deve ser de fácil acesso
  • O sistema deve permitir que os professores sugiram livros
  • O sistema deve gerar relatórios sobre quais materiais estão sendo mais utilizados
  • O sistema deve conter um painel para monitorar o uso da biblioteca

Não Funcionais

  • O sistema permite apenas professores anexarem atividades complementares aos livros
  • O sistema possibilita aos professores acompanharem o progresso dos alunos
  • O sistema permite professores indicarem leitura personalizada para os alunos considerando suas dificuldades e interesses
  • Apenas o diretor pode ter acesso ao painel de monitoramento

Inversos

  • O sistema não deve permitir que pessoas externas tenham acesso ao sistema da escola
